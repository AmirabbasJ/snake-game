# Prerequisite

- [git][git] >= 2
- [github-cli][github-cli] >= 2
- [node][nodejs] >= 14
- [npm][npm] >= 6

# how to run

first run `npm install` then run `npm run start` then the game should be running on localhost port 3000

also the code can be improved I just didn't have enough time for it
